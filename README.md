
云音乐是什么?
一个对接网易云的音乐播放器  
  * 内置集成作者自己写的歌词Lrc组件支持单行歌词和多行歌词支持悬浮窗歌词。  
  * 内置适配Android13的通知栏 （不完善请大佬请教）    
  * 对接了网易云的歌单，歌曲，搜索，二维码登录等其余功能。  
  * main.java-内api变量是网易云SDK后台服务器地址（可变更）  
  * http 是我服务器php接口的地址用于软件更新用或者其他功能（个根据需要做出必须的更改或者删除）
